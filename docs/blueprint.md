# **App Name**: StudyFlow Analytics

## Core Features:

- Pomodoro Timer: Configurable Pomodoro timer with start, pause, resume, and stop functionality, tracking subject and interruptions. Timer state persists across refreshes.
- Session Data Tracking: Stores user_id, subject_id, mode, start_time, end_time, duration, pause_count, completion_status, and calculated focus_score for each session.
- Analytics Dashboard: Displays daily, weekly, and monthly study totals, subject-wise time distribution, focus score trends, consistency streaks, peak productivity hours, and goal completion percentage. All analytics are calculated from the database records.
- Subjects Management: Allows users to create, edit, and archive subjects with optional priority level and color-coded tagging.
- Goal Setting: Enables users to set daily total study goals and weekly subject-specific goals, with automatic progress tracking and missed-goal detection.
- Session History: Provides full session logs with filtering by date, subject, and focus score. Allows editing or deleting sessions with an audit flag.
- Settings Panel: Allows configuration of Pomodoro and break durations, notification preferences, theme toggle, and data export (CSV/JSON).

## Style Guidelines:

- Primary color: Dark navy blue (#2C3E50) to evoke focus and calmness.
- Background color: Very light grey (#F0F4F8), nearly white.
- Accent color: Muted teal (#3498DB) to provide subtle contrast for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look suitable for both headlines and body text.
- Employ rounded cards and soft shadows for a modern and approachable minimal aesthetic.
- Use simple, line-based icons for navigation and key actions. Maintain consistency and readability.
- Incorporate subtle transitions and animations for a smooth user experience. Avoid excessive or distracting animations.
